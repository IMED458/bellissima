# Bellissima CRM

სილამაზისა და ესთეტიკის ცენტრი **BELLISSIMA** — CRM სისTEMა: კალენდარი, კლიენტები,
პროცედურები, თანამშრომლები, ფინანსური ანალიტიკა და შეტყობინებები.

აგებულია React 19 + Vite + Tailwind CSS v4-ზე. მონაცემები ლოკალურია (demo data),
გარე API/სერვერი არ სჭირდება.

## ლოკალურად გაშვება

**მოთხოვნა:** Node.js 18+

1. დააინსტალირე დამოკიდებულებები:
   ```bash
   npm install
   ```
2. გაუშვი dev სერვერი:
   ```bash
   npm run dev
   ```
   აპლიკაცია გაიხსნება `http://localhost:3000`-ზე.

## build / preview

```bash
npm run build     # აწყობს production build-ს dist/-ში
npm run preview   # ლოკალურად ამოწმებს აწყობილ build-ს
```
